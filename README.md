1. Introduction
Healthcare costs are increasing rapidly, making medical insurance an essential part of financial planning. This
project analyzes a dataset to predict insurance charges using machine learning techniques.
2. Problem Definition
The objective is to analyze factors affecting insurance charges, build models, and compare their performance.
3. Data Collection
Dataset: Medical Insurance Charges dataset from Kaggle with features like age, BMI, smoker status, and charges.
4. Data Preprocessing
Handled missing values, encoded categorical variables, scaled features, and split data into training and testing
sets.
5. Exploratory Data Analysis (EDA)
Performed visualization using histograms, box plots, and correlation matrices. Smoking, BMI, and age were key
factors affecting charges.
6. Feature Engineering
Selected relevant features and removed less impactful ones.
7. Model Building
Implemented Linear Regression, Decision Tree, and KNN models.
8. Model Evaluation
Evaluated using R² score and Mean Squared Error. Decision Tree performed best.
9. Conclusion
Smoking, age, and BMI significantly impact insurance costs. Machine learning models effectively predict charges.
10. Limitations
Limited dataset size and lack of real-time data.
11. Future Improvements
Use advanced models, larger datasets, and deploy as a web application.
12. Tools & Technologies
Python, Google Colab, Pandas, NumPy, Matplotlib, Scikit-learn
